# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-02-09

### Added
- Initial release of Laravel Enterprise Structure
- `make:domain` command for generating domain structures
- `make:action` command for creating reusable actions
- `make:usecase` command for orchestrating use cases
- `enterprise:install` command for installing project structure
- Configuration file for customizing paths and namespaces
- Support for PHP 8.1+ and Laravel 10.0+
- Comprehensive test suite with Pest

### Features
- Opinionated but optional project structure
- Domain-driven design patterns
- Action-based business logic organization
- Use case orchestration
- Namespace and path customization
